Как запустить сайт на GitHub Pages

1. Создай новый репозиторий на GitHub.
2. Загрузи в него файлы:
   - index.html
   - styles.css
   - папку images
3. В Settings → Pages выбери Branch: main, Folder: /root.
4. Подожди 1–2 минуты.
5. Сайт появится по адресу вида https://username.github.io/repository-name/

Как заменить картинки:
- Открой папку images.
- Замени файлы work-01.svg ... work-09.svg на настоящие фото.
- Лучше сохранять те же имена файлов, тогда ничего в коде менять не нужно.

Что стоит поменять сразу:
- В index.html блок контактов: hello@example.com и https://t.me/yourname
- При желании можно заменить текст "О мастере".
